LexLeo for macOS

Installed contents:
- /usr/local/bin/lexleo
- /usr/local/etc/lexleo/lexleo.conf
- /usr/local/share/doc/lexleo/README.txt
- /usr/local/share/doc/lexleo/LICENSE.txt
- /usr/local/share/man/man1/lexleo.1

To run LexLeo:

    lexleo

Configuration:

    /usr/local/etc/lexleo/lexleo.conf

Current supported option:

    log_path=

Logs:
- if log_path is set, the specified path is used
- otherwise the application uses its default log file path
- a probe message is written when the log file is created
