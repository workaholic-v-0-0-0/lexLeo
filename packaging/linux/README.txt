LexLeo
======

LexLeo is the LexLeo language command-line application.

Usage
-----

Run:

    lexleo

Configuration
-------------

System configuration file:

    /etc/lexleo/lexleo.conf

Current supported option:

    log_path=

If empty or omitted, the application uses its default log path.

Logs
----

The log file location depends on the configured value of:

    log_path

If no explicit path is configured, the built-in default is used.
