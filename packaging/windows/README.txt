LexLeo for Windows

Installed contents:
- bin\lexleo.exe
- config\lexleo.ini
- doc\README.txt
- doc\LICENSE.txt

To run LexLeo:
- launch bin\lexleo.exe
- or add the installation bin directory to PATH and run lexleo from a terminal

Configuration:
- edit config\lexleo.ini
- current supported option: log_path=

Logs:
- if log_path is set, the specified path is used
- otherwise the application uses its default log file path
- a probe message is written when the log file is created
